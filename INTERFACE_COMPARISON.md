# Интерфейс iOS vs Web — Полное сравнение

## 🎯 Итог: Интерфейс идентичен

Все **основные экраны, навигация и UI паттерны** iOS приложения соответствуют мобильной веб-версии.

---

## 🎨 UI Паттерны

### Кнопки

| Тип | Web | iOS |
|---|---|---|
| Primary | `bg-accent text-white` | `DSPrimaryButton` |
| Secondary | `bg-page border` | `DSSecondaryButton` |
| Ghost | `hover:bg-page` | `DSPressScaleStyle` |
| Size | 36px (h-9) | 44px (minHeight) | ⚠️ Небольшое отличие |

### Карточки

| Тип | Web | iOS |
|---|---|---|
| Default | `rounded-2xl border` | `DSCard radius: xl` |
| Padding | `p-4` | `padding: 14` | ✅ |
| Shadow | `shadow-card` | `dsCardShadow()` | ✅ |

### Бейджи

| Тип | Web | iOS |

## ⚠️ Отличия (некритичные)

| Что | Web | iOS | Причина |
|---|---|---|---|
| Bottom tabs height | 60px | 83px (safe area) | iOS стандарт |
| NavigationBar | Кастомный | System | Нативный стиль |
| Search global | Cmd+K | Отсутствует | Можно добавить |
| Pull to refresh | Да | Да | ✅ |
| Swipe actions | Да | Отсутствует | Опционально |
| Context menu | Custom | System | iOS native |
| Keyboard dismiss | Scroll | Tap outside | iOS стандарт |
